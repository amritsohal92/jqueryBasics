Concept of DOM: - It stands for document object model. - When a webpage is loaded, browser creates a Document Object Model of it. - DOM defines the standards for accessing documents. - You can think of HTML DOM as a programming mapping for HTML through which HTML elements start acting like Objects. They have their own properties, methods and events. - You should wrap all your jquery or JS code in Document Ready function. - Install jquery in page by using the CDN url. - Create document ready function. - Start writing your jQuery code inside it. $(document).ready(function(){ //write code here }); Playing with HTML using jquery - JQuery Syntax - Manipulating HTML with Jquery - Some Do's and Don'ts JQuery Syntax: - Syntax is quite literal and easy to understand. $(Selector).action(); - $ sign denotes jquery function. - Selector selects the HTML element. - . is a separator - action is to perform action on a selected element. Example below You can find elements on the basis of - Element name (like body, html, head) - Element Class - ID - Combination of class and ID - Parents and Descendents You can get and set a lot of things - GET o You can get the contents of the HTML element. o You can get the attribute values of an HTML element. - SET o You can set the contents of the HTML element o You can set the attribute values of an HTML element. - Add & remove o Add content at the beginning or end of some element. o Remove an element or an element inside it. DON'Ts - Please make sure your responsiveness is not affected. - Please make sure that the elements that have been created or introduced using jquery are not overlapping or disturbing any other elements in any screen size.
